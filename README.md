# Traceroute-on-Windows
## Install / Установка
1. Сделайте git clone данного проекта в какую то из папок.
2. Отключите брандмауэр (можно не отключать, но работоспособность в таком случае не гарантирована)
3. Запустите командную строку
4. Перейдите в папку с traceroute.exe с помощью команды cd "путь"
5. Прописывайте команду (например traceroute.exe google.com)
